* 服务一级命名：  **/topcoder**
* 管理注册中心、配置中心、Gateway、UAA（用户认证授权中心）等服务

### UAA（用户认证授权中心）

* 服务一级命名：  **/topcoder**
* 用户认证授权中心，管理用户的注册、登录、找回、信息维护等一系列

### 大Content微服务

* 服务一级命名：  **/topcoder**
* 负责静态化内容发布

### 计费微服务

* 支付业务
